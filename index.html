Java Introduction

Java is a high-level, object-oriented programming language known for its portability and robustness.

HelloWorld.java 
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Explanation: This is the simplest Java program. The main method is the entry point of any Java application. System.out.println prints text to the console.

Real-life Example

Java is widely used in enterprise applications, Android app development, and large-scale systems like banking applications.

Python Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

hello_world.py 
print("Hello, World!")

Explanation: Python programs are simple and concise. The print() function outputs text to the console. Unlike Java, no class definition or main method is required.

Real-life Example

Python is widely used in data science, web development (Django, Flask), automation, artificial intelligence, and scientific computing.
